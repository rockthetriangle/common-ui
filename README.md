## Getting Started

### Prerequisites

- **Node.js**: Version 16.x or higher
- **npm**: Version 7.x or higher (or `yarn` if preferred)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/rockthetriangle/common-ui.git
   cd common-ui	```
   
2. Install dependencies:
   ```bash
   npm install	```
   
3. Start Storybook:
   ```bash
   npm run storybook	```
